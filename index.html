Lung Disease Detection Using Chest X-ray and CNN

About the Project

This project focuses on detecting lung diseases such as Pneumonia using Convolutional Neural Networks (CNN). It uses chest X-ray images to classify whether the patient has pneumonia or not. The aim is to assist medical professionals with faster and more accurate disease identification using machine learning techniques.

Technologies Used

  • Python
  • Convolutional Neural Networks (CNN)
  • TensorFlow / Keras
  • Jupyter Notebook

Dataset

We used the publicly available Pneumonia dataset from Kaggle: Chest X-ray Pneumonia Dataset .
